# Produto

Esta pasta reúne os documentos que definem a visão, os objetivos, a filosofia e a arquitetura estratégica do projeto SAGA.

## Objetivo

Responder às seguintes perguntas:

- Por que o SAGA existe?
- Qual problema resolve?
- Quem são seus usuários?
- Quais princípios orientam seu desenvolvimento?
- Como o produto evoluirá ao longo do tempo?

Os documentos desta pasta servem como referência para todas as decisões do projeto.
